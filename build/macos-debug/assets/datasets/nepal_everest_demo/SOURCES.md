# Free Data Sources

## Terrain

- OpenTopography: https://opentopography.org/start
- OpenTopography developers: https://opentopography.org/developers
- Copernicus DEM docs: https://documentation.dataspace.copernicus.eu/APIs/SentinelHub/Data/DEM.html

Example OpenTopography bbox:

- `south=27.86`
- `north=28.12`
- `west=86.63`
- `east=87.03`

## Imagery

- Copernicus Browser: https://dataspace.copernicus.eu/browser/
- Sentinel-2 docs: https://documentation.dataspace.copernicus.eu/Data/SentinelMissions/Sentinel2.html
- Browser docs: https://documentation.dataspace.copernicus.eu/Applications/Browser.html

Recommended settings:

- Dataset: `Sentinel-2 L2A`
- Low cloud coverage
- Natural color / true color
- Export clipped image for the Everest bbox
